# CSCI-467-The-Auto-Shop

## Requirements
- Ensure you have `node.js` installed

# How to Setup
### Clone Repo 
```sh
git clone https://github.com/michaelslice/CSCI-467-The-Auto-Shop.git
```

### Setup Frontend
```sh
cd /CSCI-467-The-Auto-Shop/frontend
```

### Download Dependencies
```sh
cd npm install
npm run dev
```
---

### Setup Backend
```sh
cd /CSCI-467-The-Auto-Shop/backend
```

### Create Virtual Environment
```sh
python -m venv venv
./venv/Scripts/activate
pip install -r requirements.txt
```
### Startup Backend
Run the following command in the directory `/CSCI-467-The-Auto-Shop/backend`
```sh
python3 app.py
```

---

### View!
The frontend is available on `localhost:5173` & and the backend is on `localhost:8000`
